# NumPy Practice

Practice repo for learning and applying core NumPy concepts — array creation, indexing, broadcasting, aggregation, linear algebra, and random number generation.

## Topics Covered

- Array creation and reshaping
- Indexing, slicing, and boolean masking
- Broadcasting
- Element-wise mathematical operations
- Aggregation functions (sum, mean, max, argmax)
- Linear algebra (matrix multiplication, determinant, inverse, solving equations)
- Random number generation

## Files

- `numpy_practice.py` — Contains solutions to practice problems covering the topics above.

## How to Run

```bash
pip install numpy
python numpy_practice.py
```
